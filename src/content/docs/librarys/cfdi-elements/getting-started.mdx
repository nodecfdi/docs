Librería de elementos y/o estructuras de datos que componen un cfdi.

Librería inspirada por la versión para php https://github.com/eclipxe13/CfdiUtils modularizada.

## Primeros pasos

Está libreria depende estrictamente de `@nodecfdi/cfdi-core` por lo que es recomendable instalarla
previo a instalar la libreria cfdi-cleaner.

### Instalación

Para comenzar a utilizar la libreria lo primero que hay que hacer es instalarla, corre el siguiente
comando en tu terminal:

<Tabs syncKey="pkg">
<TabItem label="npm">

```sh
npm i @nodecfdi/cfdi-elements --save
```

</TabItem>
<TabItem label="pnpm">

```sh
pnpm add @nodecfdi/cfdi-elements
```

</TabItem>
<TabItem label="Yarn">

```sh
yarn add @nodecfdi/cfdi-elements
```

</TabItem>
</Tabs>

Con esto ya tendremos la libreria instalada y lista para ser usada.
